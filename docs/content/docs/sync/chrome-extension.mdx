## What it does

The Retrace Chrome extension syncs your AI conversations directly from the browser. No need to export files manually. It supports ChatGPT, Claude, and Gemini.

## Install the extension

1. Open Retrace and go to **Settings**
2. Click the **Chrome Extension** tab
3. Click **Install Extension** to download the `.crx` file
4. Open `chrome://extensions` in Chrome
5. Enable **Developer mode** (toggle in the top-right)
6. Drag and drop the `.crx` file onto the extensions page

## Set up syncing

1. Click the Retrace extension icon in Chrome
2. The extension will auto-detect the local Retrace server
3. Toggle on the sources you want to sync:
   - **ChatGPT** (chatgpt.com)
   - **Claude** (claude.ai)
   - **Gemini** (gemini.google.com)

## How it works

**Manual sync**: Visit a supported site, click the extension icon, and click **Sync** to import new conversations.

**Background sync**: When enabled, the extension checks for new conversations automatically when you visit a supported site. A badge shows the count of newly synced items.

## What gets synced

- New conversations since your last sync
- Full message history (your prompts and AI responses)
- Conversation titles and timestamps

## Privacy

- The extension communicates only with the local Retrace server on your machine
- No data is sent to any external service
- You control exactly which sites sync via per-source toggles
- Nothing syncs without your explicit opt-in
